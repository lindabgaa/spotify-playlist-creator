# Spotify Playlist Creator - React/Vite Project

![Project Status](https://img.shields.io/badge/Project%20Status-In%20Progress-orange)
[![CodeFactor](https://www.codefactor.io/repository/github/lindabgaa/spotify-playlist-creator/badge)](https://www.codefactor.io/repository/github/lindabgaa/spotify-playlist-creator)
[![License: MIT](https://img.shields.io/badge/License-MIT-blue)](LICENSE)

This project is an intuitive Spotify playlist creator built with **TypeScript**, **React**, and **CSS**. It uses the **Vite** build tool for fast development.
The app allows users to quickly search for songs, artists, and albums, and easily generate personalized playlists. It integrates with the [Spotify API](https://developer.spotify.com/documentation/web-api), enabling users to send their curated playlists directly to their Spotify account.
